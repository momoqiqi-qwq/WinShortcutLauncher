# v43 多开窗口与彩虹鼠标增强

本版本基于 v42 修改：

1. 修复多开新窗口白屏/空白概率：
   - Vite 构建改为相对资源路径 `base: './'`，多开窗口加载 `index.html` 时能正确找到 JS/CSS。
   - 多开窗口标题改为 `Win Shortcut Launcher - 多开`，避免 Windows `FindWindow` 唤醒主窗口时误命中多开窗口。

2. 彩虹文字新增作用范围：
   - 可选择在父目录标签生效。
   - 可选择在子目录标签生效。
   - 可选择在设置界面生效。

3. 彩虹边框增强：
   - 新增旋转/固定模式。
   - 保留动画速度调节，范围扩展到 3～120 秒。

4. 鼠标彩虹效果增强：
   - 拖尾改为参考上传 HTML 示例的定时淡出光点。
   - 新增彩点 + 光环。
   - 新增经典 Windows 箭头：边框炫彩、全炫彩、内部炫彩。
   - 新增 macOS 旋转彩虹盘。

已执行：`npm run build`。
当前环境没有 Cargo，未执行 Rust/Tauri 原生编译。
