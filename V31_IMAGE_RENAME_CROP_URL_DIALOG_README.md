# V31 图片预览改名/截取 + 添加网站主题弹窗

基于 v30 修改。

## 变更

1. 图片预览界面支持修改图片名称
   - 在图片浏览右侧预览顶部新增“修改图片名称”按钮。
   - 支持双击预览标题进入编辑。
   - 回车保存，Esc 取消。
   - 修改的是启动器图片预览内显示名称，不会直接改动磁盘原文件名。

2. 图片预览支持截取图片大小
   - 在单图操作区新增“截取图片大小”按钮。
   - 进入截取模式后，在图片上拖动选择范围，再点 ✓ 应用。
   - 截取结果保存到图片预览项的 dataUrl 中，不覆盖原始图片文件。
   - 已截取的图片可点“原”还原为原图预览。

3. 添加网站界面替换浏览器 prompt
   - 右键菜单“添加网址”不再弹出 `tauri.localhost` 原生输入框。
   - 改为软件内主题化弹窗，跟随当前主题变量：面板色、文字色、强调色、边框、阴影。
   - 支持网址为空校验、格式校验；未输入协议时自动补 `https://`。
