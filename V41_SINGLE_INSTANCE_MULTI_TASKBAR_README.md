# V41 单实例唤醒、多开按钮、贴边任务栏修正

本版基于 v40 修改：

1. 托盘后台后再次启动/点任务栏固定图标
   - 增加 Windows 单实例互斥锁。
   - 已有实例运行时，再启动 exe 不会再打开第二个主程序实例。
   - 会唤醒并聚焦已存在的主窗口；如果窗口被停放在屏幕外，会拉回屏幕中间。

2. 右上角新增多开按钮
   - 窗口控制按钮新增“多开新窗口”。
   - 每点一次会在当前进程内创建一个新的主界面窗口。
   - 多开窗口关闭时正常关闭，不影响主窗口的“关闭到托盘”。

3. 贴边隐藏时保留任务栏图标
   - Windows 原生贴边隐藏不再把主窗口真正 `SW_HIDE`。
   - 改为把主窗口滑出/停放到屏幕外但保持可见窗口样式。
   - 保留 `WS_EX_APPWINDOW`，移除 `WS_EX_TOOLWINDOW`，避免任务栏图标消失。

4. 版本更新
   - 修改版：v41
   - package / tauri 版本：0.1.41

验证：
- 已执行 `npm run build`，前端构建通过。
- 当前环境没有 Cargo，未执行 Rust/Tauri 原生编译。
