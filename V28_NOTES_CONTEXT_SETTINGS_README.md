# V28 便签右键菜单与设置折叠状态

本版基于 v27 修改：

1. 便签新增独立设置页
   - 设置 -> 便签。
   - 支持调整便签字体大小、行距、内边距、圆角、输入后保存延迟、是否自动换行、是否显示标题栏。
   - 支持配置右键分割线长度，以及横线/符号分割线使用的字符。

2. 便签编辑区右键菜单
   - 在便签文本区右键可插入横线分割线。
   - 可插入星号/自定义符号分割线。
   - 如果右键所在行为空，会直接替换当前行；如果当前行已有内容，会在下方插入一行分割线。
   - 菜单新增“剪贴板（Win+V）”，通过 Tauri 原生命令发送 Win+V 打开 Windows 剪贴板历史。

3. 箭头折叠状态
   - 搜索、中转、图片预览、便签设置页的折叠块现在默认全部收起。
   - 每个带折叠块的设置页都增加了“记住展开/收起状态”。关闭时，每次进入默认全部收起；开启后记住当前页的展开状态。
   - 界面设置页保留原有“记住展开/收起状态”，默认仍为全部收起。

4. 构建
   - 已执行 `npm run build`，前端构建通过。
   - 当前环境没有 Cargo，未在此环境执行 Rust 编译；已按现有 windows-sys 依赖新增 Win+V 原生命令。
