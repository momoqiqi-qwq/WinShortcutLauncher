# V12 变更说明

本版在 V11 基础上继续优化：

1. 左侧子目录列表区域右键也可打开“添加文件 / 添加文件夹 / 添加网址 / 添加系统功能”等空白区域右键菜单。普通右键子目录会先切换到该子目录并打开添加菜单；Ctrl/Shift + 右键仍可打开子目录管理菜单。
2. 图标加载优化：新增持久化图标缓存和并发队列，避免启动时大量 `get_file_icon` 同时调用导致卡顿。已加载过的图标会优先从本地缓存显示，再后台刷新。
3. 主窗口右上角新增最小化 `-` 和关闭 `X` 按钮。关闭行为仍遵循“隐藏到托盘 / 直接退出”设置。
4. 设置 -> 界面 -> 右上角窗口按钮 新增按钮样式设置：圆形、方形、横条，并可调节按钮大小和按钮间距。

已执行：`npm run build`。
