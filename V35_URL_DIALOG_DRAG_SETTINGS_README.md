# V35 - 添加网站弹窗 + 项目拖动设置

## 本版修改

1. 添加网站界面
   - 不再使用浏览器原生 prompt 弹窗。
   - 改为软件内主题弹窗，跟随当前主题。
   - 名称和网址放在同一个添加窗口里填写。
   - 网址未填写协议时自动补全 `https://`。

2. 设置新增“拖动”分类
   - 可设置子目录/标签页内快捷项目长按多久进入拖动。
   - 可设置拖动前允许的鼠标抖动距离。
   - 可设置拖动时项目背景颜色。
   - 可设置拖动时项目发光颜色和发光亮度。
   - 设置页内提供拖动效果预览。

3. 项目拖动效果
   - 拖动中的项目会使用用户设置的背景色与光晕。
   - 单击启动模式下继续保留“点击项目只打开，不吸附到鼠标”的修复。

## 构建验证

已执行：

```bash
node node_modules/typescript/lib/tsc.js
node node_modules/vite/bin/vite.js build
```

前端构建通过。当前环境未执行 Rust/Tauri 原生编译。
