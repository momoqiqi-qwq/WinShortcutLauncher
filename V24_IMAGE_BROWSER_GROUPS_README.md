# V24 图片浏览分组与自定义设置

本版更新：

1. 图片浏览器支持分组
   - 图片浏览器顶部新增分组栏。
   - 可新增分组、重命名当前分组、删除当前分组。
   - 拖入或添加图片会进入当前选中的分组。
   - 删除分组时，该分组内图片会移动到剩余分组，避免误删图片记录。

2. 图片浏览器左边缘拖动范围放宽
   - 面板左边缘可拖动到更大或更小范围。
   - 缩略图区和预览区之间的分割条也放宽范围。

3. 图片名称位置可配置
   - 支持：图片下方、图片上方、图片内部、不显示。
   - 可在图片浏览器右上角设置中修改，也可在“设置 → 搜索/中转 → 图片浏览”中修改。

4. 图片浏览按钮显示可配置
   - 可显示/隐藏“添加图片”“复制本组到文件夹”“清空本组”。
   - 可显示/隐藏单张图片的复制路径、打开、复制到文件夹、移除等操作按钮。

已通过前端构建：npm run build。
