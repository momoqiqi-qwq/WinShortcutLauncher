# V30 - 操作设置折叠与主题去重

基于 v29 修改。

## 修改内容

1. 设置 - 操作 添加箭头折叠
   - 将“操作行为”拆成三个折叠块：
     - 贴边隐藏与动画
     - 贴边触发条显示
     - 启动与关闭
   - 默认进入时全部折叠。
   - 增加“记住展开/收起状态”、全部收起、全部展开。
   - 折叠状态使用 localStorage 保存，不影响原配置结构。

2. 主题重复修复
   - “外观”区域只显示基础内置主题。
   - “更多主题”区域只显示 MORE_THEMES。
   - 修复夜蓝、石墨黑、玻璃蓝等更多主题在两个主题区域重复出现的问题。

## 验证

已执行：

```bash
npm ci
npm run build
```

前端构建通过。
