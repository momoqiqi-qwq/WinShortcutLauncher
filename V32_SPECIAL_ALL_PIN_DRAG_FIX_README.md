# V32 - 全部特殊标签、置顶钉子、标签点击拖拽修复

## 本次修改

1. **“全部”特殊标签添加子标签拦截**
   - 在“全部”标签界面点击左侧新增子目录按钮时，不再新增标签。
   - 直接在主界面底部显示红色提示：`特殊标签不可添加子标签`。
   - 通过标签右键菜单在“全部”标签上新增标签时也会拦截并提示。
   - 同一父目录下重复新建“全部”特殊标签时会提示：`特殊标签已存在`。
   - 特殊标签拖拽排序会提示：`特殊标签不可排序`。

2. **右上角新增置顶钉子**
   - 右上角控制按钮新增“窗口置顶/取消置顶”钉子。
   - 置顶状态会写入配置，重启后保留。
   - 按钮仍支持和其它右上角按钮一样拖动调整顺序。

3. **修复单击启动模式下点击标签被吸附到鼠标的问题**
   - 父目录标签、子目录标签改为 `data-no-drag`，避免触发窗口拖动。
   - 标签排序拖拽阈值从 6px 提高到 14px，降低普通点击误触发拖拽概率。
   - 标签组件改为手动转发 dnd-kit pointer 事件，先阻止窗口拖动，再允许真正拖拽排序。
   - “全部/便签”等特殊标签禁用排序拖拽。

- 结果：前端构建通过。
