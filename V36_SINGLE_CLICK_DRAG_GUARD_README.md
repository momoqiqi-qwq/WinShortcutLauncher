# V36 单击启动误触拖拽修复

本版针对视频中“单击子目录项目后，项目偶发吸附到鼠标上”的情况做了更严格的拖拽防误触处理。

## 修复内容

1. 单击启动模式下，项目不再在 pointerdown 时立即交给 dnd-kit。
2. 只有鼠标按住达到“设置 - 拖动”里的长按时间后，才会把该项目转交给排序拖拽逻辑。
3. 单纯慢一点的点击不会自动进入拖拽态；必须长按后再移动，才会开始拖动排序。
4. 单击打开项目时，启动外部程序延后 60ms，让 dnd-kit 先处理完 pointerup 清理，避免外部程序抢焦点时遗留拖拽状态。
5. 保留 v35 的拖动设置项：长按时间、移动容差、拖动背景、发光颜色和亮度。
