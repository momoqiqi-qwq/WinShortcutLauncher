# V11 设置折叠与启动修复

## 改动

1. 设置 -> 界面 默认进入时所有设置块全部收起，和截图中的默认状态一致。
2. 新增「记住展开/收起状态」开关：
   - 关闭：每次进入界面设置都恢复默认全部收起。
   - 开启：保留你上次展开或收起的状态。
3. 新增「全部收起」「全部展开」快捷按钮。
4. 修复单击启动/双击启动事件冲突：
   - 单击启动使用 pointer up 判定，拖拽、Ctrl 多选、Shift/Alt 不会误启动。
   - 双击启动只在双击时启动，单击只选中。
   - 增加 280ms 启动锁，避免一次双击或事件冒泡导致重复启动。

npm install
npm run tauri:dev
