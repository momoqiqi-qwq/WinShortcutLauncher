# V13 - 四宫格窗口按钮与自定义背景

本版新增：

1. 设置 -> 界面 -> 右上角窗口按钮：新增「四宫格」显示方式，对应 2x2 控制面板样式。
2. 设置 -> 界面 -> 背景图片：支持选择本地图片或手动输入图片路径。
3. 背景常用参数：启用/关闭、透明度、压暗、模糊、填充方式、位置、面板不透明度。
4. 背景设置会自动随现有 Zustand 配置持久化，旧配置会自动补齐默认值。

Windows 运行：

```powershell
npm install
npm run tauri:dev
```

构建 exe：

```powershell
npm run tauri:build
```
