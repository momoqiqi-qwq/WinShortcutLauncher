# V15 更新说明

本版本在 V14 基础上新增与修复：

1. 父目录标签不再限制 3 行。等宽标签开启后，新增父目录会自然扩展到更多行，不再强制横向滚动。
2. 右上角新增全局搜索按钮。可搜索所有普通标签中的项目，支持定位到所属父目录/子目录，也支持直接打开。快捷键：Ctrl+K。
3. 右上角新增文件中转站。可临时加入文件/文件夹，复制路径、打开、添加到当前普通目录或清空。
4. 支持合并父目录与子目录：
   - 父目录右键：合并到其他分区。
   - 子目录管理菜单：合并到同类型标签。
   - 内容区/子目录区域右键：可对当前标签执行合并。
5. 空普通子目录与便签可自由切换：
   - 空普通标签可以切换为便签。
   - 便签可以切换为普通标签，原便签内容保留在配置里，切回便签时仍可见。
