# V40 favicon 复用与图片截取修正

## 修改内容

1. 本地 favicon 复用
   - 新增 `get_cached_website_favicon` Tauri 命令。
   - 添加网站或刷新网址图标时，若本地已经下载过同一网站或其子域名/父域名 favicon，会优先复用本地缓存。
   - 没有缓存时才继续下载 favicon。

2. 图片预览截取修正
   - 取消覆盖界面的大十字准线。
   - 保留系统鼠标小十字光标。
   - 截图框使用鼠标实际位置作为右下角，修正右下角跟随偏移。
   - 兼容预览区域滚动后的坐标计算。

3. 版本号
   - 修改版：v40
   - package / tauri 版本：0.1.40

## 构建

已执行：

```bash
npm run build
```

前端构建通过。当前环境没有 Cargo，未执行 Rust/Tauri 原生编译。
