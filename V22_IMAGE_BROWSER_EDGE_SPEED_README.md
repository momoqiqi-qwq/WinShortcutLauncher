# V22 图片浏览器与贴边隐藏速度

本版新增：

1. 贴边展开后，鼠标移出主界面时的隐藏动画速度可独立调节。
   - 位置：设置 -> 操作 -> 鼠标移出主界面后隐藏速度
   - 字段：behavior.edgeMouseLeaveHideMs
   - 会透传到 Rust 原生贴边控制器 NativeEdgeOptions.mouseLeaveHideMs

2. 右上角新增“图片浏览”按钮。
   - 支持拖入 png/jpg/jpeg/webp/gif/bmp/svg/ico/tif/tiff/avif 图片。
   - 左侧缩略图列表，右侧大图预览，布局参考幻灯片浏览器。
   - 可拖动缩略图或预览图到外部文件夹；如果 Windows 资源管理器不接收 WebView 拖出数据，可以使用“复制到文件夹”。

3. 设置 -> 搜索/中转 新增“图片浏览”设置。
   - 面板宽度
   - 缩略图宽度
   - 预览适配方式：完整显示 / 填满区域 / 原始大小
   - 是否允许拖入图片
   - 拖出/复制动作：复制 / 移动
   - 清空前确认

4. 拖入行为隔离。
   - 图片浏览器打开时，外部拖入只进入图片浏览器，不会同时触发添加快捷项目。
   - 文件中转站逻辑保持不变。

前端构建已测试通过：npm run build。
Rust/Cargo 需要在 Windows 本机验证。
